title: Docker
description: Guia de aprendizaje docker
---

Docker es un sistema de contenedores que utiliza el kernel de Linux para aislar procesos sin necesidad de máquinas virtuales.

Un contenedor se refiere a un entorno aislado de desarrollo y ejecución para alojar una aplicación individual.

El contenedor engloba código fuente, dependencias, librerías, configuración, runtime (node, python, .net)

Se utiliza mucho para desarrollar para que tu entorno sea idéntico al del resto del equipo.

Cuando se crea un nuevo contenedor hay que instalarle su lenguaje de programación, su versión de package manager y demás, para que de esa forma cuando compartas ese contenedor y se ejecute en otra computadora entonces trabaje exactamente igual.
